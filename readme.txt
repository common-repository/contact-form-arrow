== Installation ==

**The easy way** :

* Go to the Plugins Menu in WordPress.
* Search for plugin "Arrow Contact Form"
* Click "Install".
* After Installation click activate to start using the Contact Form.
* Go to "CFA Contact form" from Dashboard menu.



If you are having difficulties with installation please contact us [Support Forum](https://www.arrowplugins.com/support/)
